# Справочник погрузчиков

Веб-приложение для управления справочником погрузчиков и учёта простоев.

## Стек технологий

| Слой | Технологии |
|------|-----------|
| База данных | PostgreSQL 16 |
| Backend | Python 3.12, FastAPI, SQLAlchemy 2.0, Alembic |
| Frontend | React 18, TypeScript, Vite, Ant Design 5 |
| Инфраструктура | Docker, Docker Compose |

## Быстрый старт (Docker Compose)

```bash
# Клонировать / открыть директорию проекта
cd /path/to/project

# Запустить все сервисы (БД + бэкенд + фронтенд)
docker compose up --build
```

После запуска:
- **Фронтенд**: http://localhost:5173
- **Swagger UI (API)**: http://localhost:8000/docs
- **ReDoc**: http://localhost:8000/redoc

## Локальная разработка

### Требования

- Python 3.12+
- Node.js 20+
- PostgreSQL 16 (или Docker)

### Backend

```bash
cd backend

# Создать виртуальное окружение
python -m venv .venv
source .venv/bin/activate       # Windows: .venv\Scripts\activate

# Установить зависимости
pip install -r requirements.txt

# Скопировать конфиг БД
cp .env.example .env
# При необходимости отредактировать DATABASE_URL

# Применить миграции
alembic upgrade head

# Запустить сервер
uvicorn app.main:app --reload --port 8000
```

### Frontend

```bash
cd frontend

npm install
npm run dev
```

## Функциональность

### Справочник погрузчиков

| Функция | Описание |
|---------|----------|
| Поиск | По номеру погрузчика (вхождение строки, без учёта регистра) |
| Добавить | Новая строка в таблице для ввода данных |
| Изменить | Перевод записи в режим редактирования |
| Удалить | С подтверждением; запрещено при наличии простоев |
| Сохранить | Сохранение изменений в БД |
| Отменить | Отмена с подтверждением |

### Реквизиты погрузчика

- **Код** — автоинкрементный идентификатор
- **Марка** — строка
- **Номер** — строка
- **Грузоподъёмность** — число, до 3 знаков после запятой
- **Активен** — признак действующей записи (по умолчанию `true`)
- **Дата изменения** — заполняется автоматически
- **Пользователь** — ФИО пользователя, проводившего редактирование

### Простои по погрузчику

Отображаются для выбранной строки таблицы погрузчиков. Сортировка по умолчанию: по дате начала, обратный порядок.

| Функция | Описание |
|---------|----------|
| Добавить инцидент | Модальное окно; дата начала заполняется текущим временем |
| Изменить инцидент | Модальное окно с предзаполненными данными |
| Удалить инцидент | С подтверждением |

**Время простоя** рассчитывается автоматически: дата окончания − дата начала (если окончание не задано — до текущего момента). Отображается в формате «Xч Yмин».

## API Endpoints

| Метод | URL | Описание |
|-------|-----|----------|
| GET | `/forklifts/` | Список погрузчиков (опц. фильтр `?number=...`) |
| GET | `/forklifts/{id}` | Погрузчик с инцидентами |
| POST | `/forklifts/` | Создать погрузчик |
| PUT | `/forklifts/{id}` | Обновить погрузчик |
| DELETE | `/forklifts/{id}` | Удалить погрузчик |
| GET | `/forklifts/{id}/incidents/` | Простои погрузчика |
| POST | `/forklifts/{id}/incidents/` | Создать инцидент |
| PUT | `/forklifts/{id}/incidents/{inc_id}` | Обновить инцидент |
| DELETE | `/forklifts/{id}/incidents/{inc_id}` | Удалить инцидент |
